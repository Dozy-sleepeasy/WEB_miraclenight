공지사항

📬 여러분의 목소리를 들려주세요!
미라클나잇은 여러분께 세계 최고의 수면 경험을 제공하기 위해 최선을 다하고 있습니다. 서비스의 부족한 점을 알려주시면 빠르게 고치고 더 좋은 모습으로 찾아오겠습니다. 의견을 전달하고 싶으신가요?

Version 1.4.1 기능 업데이트 new

🔥 새로운 명상으로 업데이트 되었습니다
바디스캔 명상이 추가됐습니다. 흔들리는 풀밭에 위에 누워 나의 몸 구석구석에 주의를 기울이며 숙면을 준비해보세요.

🔥 수면 캘린더가 업데이트 되었습니다
열람하지 않은 날짜를 표시했으면 좋겠다는 의견을 토대로 캘린더가 업그레이드 되었습니다. 열람하지 않은 날짜는 파란색 점이 생기고, 수면 데이터가 없는 날짜는 클릭이 불가능합니다. :)

Version 1.2.3 기능 업데이트

🔥 모노럴비트와 ASMR 믹싱 기능이 추가되었습니다
모노럴비트만 듣는 것이 어려웠던 분들을 위해 ASMR과 함께 들을 수 있는 믹싱 기능이 추가되었습니다. 좋아하는 ASMR 사운드와 함께 믹싱하여 마음을 진정하고 더 편안한 상태로 수면에 들어보세요.

🔥 5가지 ASMR 사운드가 추가되었습니다
전문 사운드 디자이너 저스틴이 제작한 ASMR 사운드가 5개 추가되었습니다.
- 파도 위에서 마주한 수평선
- 깊은 바다의 심장소리
- 향기로 가득찬 어둑한 우림
- 모두가 잠에 든 밤 비행기
- 비워내는 밤의 상념

Version 1.2.1 기능 업데이트

🔥 아카이빙 페이지가 업그레이드 되었습니다
캘린더 베이스로 더욱 보기 편하게 디자인이 변경되었습니다. 자기 전 내가 선택한 상태와 일어나서 나의 피드백을 한 눈에 확인하세요!

🔥 자동 종료 기능이 추가되었습니다
앱으로 돌아와 수동으로 수면을 종료하지 않아도 자동으로 수면이 종료됩니다. 편하게 아침에 일어나 하루를 시작하세요!

🔥 이메일 회원가입 기능이 추가되었습니다
애플 이메일이 아닌 일반 이메일로도 회원가입 및 로그인을 할 수 있는 기능이 추가되었습니다. 제휴 기관을 통해 오는 회원분들께서는 이메일로 회원가입하여 혜택을 누리시길 바랍니다 :)

🔥 알람 기능과 뒤척임 기능이 서비스 종료되었습니다
미라클나잇에서 제공하는 수면 서비스가 더욱 명확한 방향으로 성장하기 위해 알람 기능과 뒤척임 기능이 종료되었습니다. 이후 서비스가 업그레이드 된 후 더욱 안정적으로 도입될 예정이니 많은 관심과 의견 부탁드립니다.

Version 1.1.1 기능 업데이트

🔥 상태에 맞는 모노럴비트 믹싱 기능
오늘 나의 상태를 선택하면 최적화 인공지능 알고리즘으로 나에게 딱 맞는 모노럴비트 믹스를 추천합니다! 맞춤화된 모노럴비트 사운드로 더욱 최적화된 수면을 경험해 보세요.

🔥 수면 기록 기능
오늘 나의 수면이 어땠는지 피드백하여 수면 데이터를 쌓고 나만을 위한 알고리즘을 고도화 해보세요! 기상 시 기분과 한 줄 기록으로 간편하게 수면 상태를 기록해 보세요.

Version 1.0.57 기능 업데이트

🔥 수면 진입 명상 기능
수면에 들어갈 때 도움이 될 수 있는 명상이 추가되었습니다. 연세대학교 심리학과 박사가 직접 작성한 스트레스 해소 명상으로 우주 공간에서 무중력에 몸을 맡기고 호흡에 집중하며 숙면을 준비해 보세요.

🔥 이용 백서
미라클나잇을 100배 더 잘 활용할 수 있는 이용 백서가 추가되었습니다. 수면 준비 단계부터 수면 중, 수면 후, 결제 그리고 문의까지 간단한 가이드를 확인하여 미라클나잇을 더 쉽게 사용해 보세요.

🔥 모노럴비트의 뇌과학 원리 매거진
모노럴비트가 어떻게 수면을 유도하는지에 대한 뇌과학적 해석을 설명해 드립니다. 미라클나잇만의 레이어드 모노럴비트가 어떻게 여러분의 수면을 돕고 있는지 확인해 보세요.

🔥 자주 묻는 질문
서비스를 이용하시며 생길 수 있는 질문 12가지에 대한 명쾌한 해답을 추가하였습니다. 모노럴비트, 볼륨 조절, 배터리 및 발열 그리고 결제까지 자주 묻는 질문 12가지에 대한 답변을 확인해 보세요.

Version 1.0.51 기능 업데이트

Version 1.0.40 기능 업데이트

Version 1.0.40 기능 업데이트

🔥 일시정지 기능
수면 중 잠시 중단해야 하는 상황이 생길 수 있어 일시정지 기능을 추가하였습니다. 다만 미라클나잇이 오랜 시간 백그라운드에서 작동할 경우 강제 종료되어 처음부터 다시 시작할 수 있으니 너무 오랜 시간 동안 백그라운드로 돌려놓는 것은 삼가해주시길 바랍니다 :)

Version 1.0.36 기능 업데이트

🔥 수면 중 뒤척임 측정 기능
수면 중 고객님의 상태를 파악하고 수면 유도 소리의 효과가 고객님께 어떤 영향을 드렸는지 객관적으로 파악하기 위해 뒤척임 감지 기능을 추가하였습니다. 매트리스 위에 핸드폰을 올려놓고 주무시면 수면 중 몇 번의 뒤척임이 있었는지 분석하여 알려드립니다.

Version 1.0.30 기능 업데이트

🔥 알람 기능
고객분들께서 4시간 30분 후에 알람 없이 일어나기가 어렵다는 피드백을 많이 주셔서 알람 기능을 추가하였습니다. 수면 중 화면에서 알람 ON/OFF를 설정하여 원하시는 날에 4시간 30분 후 알람을 받아보실 수 있습니다. 알람은 소리와 진동이 함께 울립니다.

🔥 수면 종료 버튼
수면 후 수면을 고객님이 직접 종료하는 기능을 추가하였습니다. 수면 트레킹이 시작되면 수면 시작과 종료 시간을 직접 조절하여 정확한 수면 상태 파악 및 분석을 진행해야 하기 때문에 추가하였습니다. 4시간 30분 수면 소리 재생 이후 화면의 종료 버튼을 길게 눌러 수면을 종료하세요.
